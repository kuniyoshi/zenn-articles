# 自分の人生に大きな影響を与えた本

私が大きな影響を受けた本を、時系列順に紹介します。

- 理科系の作文技術、木下是雄、中公新書
- 初めての Perl、Randal L. Schwartz, Tom Phoenix、オライリージャパン
- SOFT SKILLS ソフトウェア開発者の人生マニュアル、ジョン・ソンメズ、日経BP

# 理科系の作文技術

本当に一生モノの技術だと思います。「この本を紹介しておいて、この程度の作文技術かよ」ってなりそうなので紹介するプレッシャーが大きいです。

「理科系の作文技術」はテクニカルライティングの本です。書き手の意図を読み手に伝えるための書き方の本です。

初めてこの本を読んだのは大学時代の講義です。それまでは作文といえば感想文でした。実はこの頃は文章を書くのが苦痛でした。文才を評価されるような気分だったのです。しかし、「理科系の作文技術」で習ったのはマニュアル的な作文方法でした。数学の公式を習ったような感じがしました。つまり、これは文章を書く方法を使っただけで自分の文才を使ったわけじゃない、という大義名分により文章を書く苦痛が減りました。

以来、「理科系の作文技術」は人生の窮地を何度も救ってくれました。最初は新卒の就職活動です。全然うまくいかなくて、そういえばと思い出してこの本を読みながら履歴書を直しているとようやく就職できました。二回目は資格試験です。情報処理技術者試験にハマっていた時期に午後の点数が取れずに困っていました。解説を読んでも同じことを書いたつもりなのでなんで合格できなかったのかわからなかったです。またしてもそういえばと思い出して、この本を何回か読みながら自分の回答と、解説とを読み比べることを続けてようやく合格できました。三回目は転職活動です。不採用が続いていたのでそういえばと思ってこの本をんで応募書類を書き直していたらついに採用されました。なんだか怪しいマーケティングみたいな体験録ですが、そのくらいこの本には助けられました。都合、10 回くらいは読んでいると思います。

古い本ではあるので、今の時代にあったものがあるかもと思って何冊か読んでみましたが、やっぱり理科系の作文技術がいいと思います。流石に OHP (古いプロジェクター) の部分はもう役に立たないとは思いますけども。

# 初めての Perl

「初めての Perl」が大きな影響力を持った要因はタイミングにありそうなので、他の人には刺さらないと思います。

[別の記事](https://zenn.dev/pura/articles/c8814c15557619)でも言及しましたが、私がこの本を読んだときは 1 ページを読むのに 30 分とか 1 時間とかかかることがありました。そのくらいの苦労をして読んだ本なら人生に大きな影響を与えるのも納得のことです。

「初めての Perl」はーーそれでも読むのに時間がかかりすぎたけどーー内容の分かり易さとユーモアによる楽しさとがあって読み切ることができました。この本以前の勉強は教科書や情報技術者試験の参考書などでした。「初めての Perl」以降はオライリーの本を好んで読むようになるんですが、オライリーの本は内容の分かり易さと文章の分かり易さ、そしてちょっとしたユーモアもあって学習しやすいです。オライリーの本を読み始めて感じた衝撃は大きかったです。海外で出版されて翻訳されるのは上澄のものだけとはいえ読みやすさが段違いだと思います。

「初めての Perl」でプログラマーを始めたのと、オライリーの本を色々読み始めたのとがこの本の影響力です。

# SOFT SKILLS ソフトウェア開発者の人生マニュアル

すごい人がどういう行動をしているかがわかって勉強になりました。

面接や交渉のやり方もためになりました。有料の履歴書の添削サービスや印象的な応募など新しい発見がありました。

運動や料理の話題など説得力がありました。

中でも大きかったのはポモドーロテクニックです。何度か耳にしてましたが、この本を読んでからやってみようと始めました。実際、私にはとても合っていました。以前はデバグ中や実装中に時間が経つのがあっという間で効率的に仕事できているつもりでした。しかし、ポモドーロテクニックを始めてみると時間は過ぎていても効率はよくなかったと気づきました。以前より多くのことができるようになったのです。合間に休んでいるにもかかわらず。

本で紹介されていた株式投資も始めました。

本の中で紹介されていた本もいくつか読みました。いい本は、新しい行動を促しますよね。ポモドーロテクニックもそうですけど、紹介されていた下記の本を読みました。習得への情熱やリファクタリングなど SOFT SKILLS 以降のモチベーションが高かったです。

# 終わりに

何回も読むようないい本との出会いはいいものです。「初めての Perl」は今は勧めづらいですが、「理科系の作文技術」と「SOFT SKILLS ソフトウェア開発者の人生マニュアル」とは今でもオススメの本です。これから転職活動になるので SOFT SKILLS は再読する予定です。応募書類作成時には理科系の作文技術も読むかもしれません。
