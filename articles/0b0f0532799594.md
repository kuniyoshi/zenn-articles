# 自分の人生に大きな影響を与えた本

私が大きな影響を受けた本を、時系列順に紹介します。

- 理科系の作文技術、木下是雄、中公新書
- 初めての Perl、Randal L. Schwartz, Tom Phoenix、オライリージャパン
- SOFT SKILLS ソフトウェア開発者の人生マニュアル、ジョン・ソンメズ、日経BP

それぞれ、ピンチを何度も救ってくれた本、初めて勉強できた本、広がりを持たせてくれた本です。

# 理科系の作文技術

作文は本当に一生モノの技術だと思います。「この本を紹介しておいて、この程度の作文技術かよ」ってなりそうなので紹介するプレッシャーが大きいです。

「理科系の作文技術」はテクニカルライティングの本です。書き手の意図を読み手に伝えるための書き方の本です。

初めてこの本を読んだのは大学時代の講義です。それまでは作文といえば感想文でした。実はこの頃は文章を書くのが苦痛でした。文才を評価されるような気分だったのです。しかし、「理科系の作文技術」で習ったのはマニュアル的な作文方法でした。数学の公式を習ったような感じがしました。つまり、「これは文章を書く方法を使っただけで自分の文才を使ったわけじゃない」、という大義名分により文章を書く苦痛が減りました。

以来、「理科系の作文技術」は人生の窮地を何度も救ってくれました。最初は新卒の就職活動です。全然うまくいかなくて、そういえばと思い出してこの本を読みながら履歴書を直しているとようやく就職できました。二回目は資格試験です。情報処理技術者試験にハマっていた時期に午後の記述問題の点数が取れずに困っていました。解説を読んでみても同じことを書いたつもりなので合格できなかった理由がわかりませんでした。またしてもそういえばと思い出して、この本を何回か読みながら自分の回答と、解説とを読み比べていると、言いたいことが書けてないと気づきました。三回目は転職活動です。不採用が続いていたのでそういえばと思ってこの本を読んで応募書類を書き直していたらついに採用されました。なんだか怪しいマーケティングみたいな体験録ですが、そのくらい、この本には何度も助けられました。都合、10 回くらいは読んでいると思います。

古い本ではあるので、今の時代にあったものがあるかもと思ってテクニカルライティングの本を何冊か読んでみましたが、やっぱり理科系の作文技術がいいと思います。流石に OHP (古いプロジェクター) の部分はもう役に立たないとは思いますけども。

# 初めての Perl

「初めての Perl」では勉強を楽しめること、難しくても意外と分かるようになることを体験できました。

[別の記事](https://zenn.dev/pura/articles/c8814c15557619)でも言及しましたが、私がこの本を読んだときは 1 ページを読むのに 30 分とか 1 時間とかかかることがありました。そのくらいの苦労をして読んだ本なら人生に大きな影響を与えるのも納得のことです。

大学の卒業研究では C 言語で画像処理のコードを書いていたので、多少はプログラミングが分かると思っていました。しかしブランクのせいか、言語の違いのせいか内容が理解できなかったです。最初は分からないところを飛ばして進めようとしましたが、飛ばすところが増える一方で、これはダメだと最初から読み直しました。今度は分からないところを分かるまで何回も読みながら進めました。なんとか読み終わる頃にはちょっとは perl が分かるようになってました。そのあとで、もう一回最初から読みました。「初めての Perl」の読書体験は初めて自分で**勉強**できたような出来事でした。

内容には苦労しましたが、読み終えてみるとちゃんと分かるように文章が書かれていたように思います。以降、オライリーの本を好んで読むようになるのでそこからも影響力が大きかったのがわかります。

# SOFT SKILLS ソフトウェア開発者の人生マニュアル

この本のように人生の本となると内容が多岐にわたってしまいます。その中の多くに説得力を感じ、共感し、そして行動できました。

著者がソフトウェア開発者なので、プログラマーがちょうど興味を持たなそうなことを知っていてそれを扱っているためどの内容も面白かったです。面接や給与等交渉のやり方、運動、料理、投資などです。私は転職活動で苦労している方なので、有料の履歴書の添削サービスや印象的な応募などの新しい発見も面白かったです。

中でも大きかったのはポモドーロテクニックです。それまでも何度か耳にしてましたが、この本を読んでからやってみようと始めました。やってみると私にはとても合っていました。以前からデバグ中や実装中に時間が経つのがあっという間で、効率的に仕事できているつもりでした。しかし、ポモドーロテクニックを始めてみると時間は過ぎていても効率はよくなかったと気づきました。時間を区切り始めて、以前より多くのことができるようになったのです。合間に休んでいるにもかかわらず。

本の中で紹介されていた本もいくつか読みました。いい本は、新しい行動を促しますよね。ポモドーロテクニックを始めたこともそうですけど、紹介されていた下記の本を読みました。習得への情熱やリファクタリングなどもこの頃に読んでいて SOFT SKILLS 以降のモチベーションが高かったです。

# 終わりに

人生を変えるような本は多くはありません。しかし、何度も読み返すような本は人生に大きな影響を与えます。

私にとってはこの 3 冊がそうでした。
