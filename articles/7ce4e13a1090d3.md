Unity プロジェクトで開発しているとき、UnityEngine.Object の null 比較にヒントが出ているのを見つけました。null 比較がオーバーライドされているからですが、「これってエラーにしないと見逃すリスクの方が大きいのでは?」と思いました。

[Microsoft.Unity.Analyzers](https://github.com/microsoft/Microsoft.Unity.Analyzers?tab=readme-ov-file) に多くの定義があったので全部確認して設定しました。

```
none
silent
suggestion
warning
error
```
